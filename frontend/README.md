# Project Management Portal

A full-stack task management application built using React.js and Node.js.

## Features

- View Tasks
- Add New Task
- Complete Tasks
- Delete Tasks
- Filter Tasks by Status
- Responsive Dashboard UI

## Tech Stack

### Frontend
- React.js
- Axios
- React Router DOM
- CSS

### Backend
- Node.js
- Express.js
- CORS

## Installation

### Backend

```bash
cd backend
npm install
node server.js
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## API Endpoints

### Get All Tasks
GET /tasks

### Add Task
POST /tasks

### Complete Task
PUT /tasks/:id

### Delete Task
DELETE /tasks/:id
